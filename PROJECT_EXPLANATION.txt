PROJECT TITLE:
LINKGUARD-AI – Explainable AI-Powered URL Phishing & Security Risk Detection System


====================================================
1. OVERALL PROJECT OVERVIEW
====================================================

LINKGUARD-AI is a full-fledged URL security scanner that detects phishing and malicious URLs using a hybrid approach combining:
- Machine Learning
- Rule-based security analysis
- Explainable AI (XAI)

Unlike traditional blacklist-based systems, LINKGUARD-AI analyzes the FULL URL (domain, path, query) to detect modern phishing techniques, including AI-generated phishing URLs.


====================================================
2. FLOW OF EXECUTION (END-TO-END)
====================================================

1. User enters a URL in the frontend UI (index.html)
2. Flask backend (app.py) receives the request
3. Features are extracted from the FULL URL
4. Trained ML model predicts phishing probability
5. Hybrid Risk Fusion Engine computes final risk
6. Explainable AI generates human-readable explanation
7. Risk result is returned to UI and displayed

FLOW:
User → UI → Flask API → Feature Extraction → ML Prediction
→ Hybrid Risk Fusion → Explainable AI → UI Output

def detect_brand_impersonation(url):
    parsed = urlparse(url)
    domain = parsed.netloc.lower()
    for brand in BRANDS:
        if brand in url.lower() and brand not in domain:
            return 1
    return 0

def shannon_entropy(s):
    probs = [float(s.count(c)) / len(s) for c in set(s)]
    return -sum(p * math.log2(p) for p in probs)
